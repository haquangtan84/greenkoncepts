<h1>Usage</h1>

Install Node.JS and using npm the packages :
```
npm install <package>
```
<ul>
  <li>bcrypt</li>
  <li>everyauth</li>
  <li>express</li>
  <li>mongodb</li>
  <li>oauth</li>
</ul>

Then, run the following command :
```
node main.js
```
